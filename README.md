# Compiler

Учебный компилятор на C++23.

## Структура

- `src/main.cpp` - точка входа
- `src/driver` - разбор CLI, чтение входного файла и запуск пайплайна
- `src/frontend` - лексер и будущий синтаксический фронтенд
- `docs/grammar.md` - текущая грамматика языка

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Запуск

```bash
./build/compiler [options] <input_file>
```

Доступные флаги:

- `-h`, `--help`
- `-v`, `--version`
- `--show-tokens`
- `--show-ast`
- `--show-ir`
- `--show-asm`

## Состояние проекта

На текущем этапе реализованы:

- базовая структура проекта
- загрузка исходного файла
- лексический анализ
- вывод токенов и лексических ошибок
